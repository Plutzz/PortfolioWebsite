Harold's Perplexion

Unity 2D | Brackeys Game Jam 2024.1

Description

A 2D Puzzle Platformer. Push boxes, flip levers, and open doors into other dimensions to help Harold discover the secrets of the magical world that he has found himself in.

Contributions

  • Produced a game that placed in the top 30 in a game jam with over 750 project submissions.
  • Developed a custom camera transition system that keeps track of when and where to transition the camera once a player enters a trigger.
  • Implemented a level system that allowed for the easy creation of multiple levels using a custom C# script and Unity Scenes.
  • Utilized C# interfaces to create multiple different interactable puzzle elements.
